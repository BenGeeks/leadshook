Functions

Function Name Description Example
DT.setField Set custom field value. DT.setField('first_name', 'John')
DT.grabCookie Get cookie value. DT.grabCookie('fbc', '_fbc')
Now field {fbc} should have value from _fbc cookie. E.g. fb.1.1554763741205.AbCdEfGhIjKlMnOpQrStUvWxYz1234567890
DT.setCookie Set cookie. DT.setCookie('lead_id', '{lead_id}', 30)
Set cookie with name lead_id and use {lead_id} as its value with expiry 30 days from now. Please note that expiry days is optional.
